# Icons

Placeholder SVG icons are provided. Replace with proper PNG icons for best PWA compatibility.

## Required Icons

| File | Size | Format |
|------|------|--------|
| `icon-192.png` | 192×192 | PNG |
| `icon-512.png` | 512×512 | PNG |

## Generating PNG Icons

1. Open the SVG files in any image editor (Figma, Illustrator, Inkscape, etc.)
2. Export at 192×192 and 512×512 as PNG
3. Update `manifest.webmanifest` to reference the PNG files instead of SVG

## Icon Design

The placeholder icon is a gold star on a dark green background — evoking a treasure map / adventure theme.

For production, consider a compass, treasure chest, or map pin icon in gold on dark green.
